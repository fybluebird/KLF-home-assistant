# 🏠 家庭助手

> 全家人都能用的智能语音助手

## 功能特点

- 🎤 语音对话 - 说话就能用
- 👴👶 简单易用 - 5岁到80岁都能用
- 🌐 网页界面 - 浏览器直接访问

## 快速开始

### 1. 启动服务

```bash
cd home-assistant
python3 voice_server.py
```

### 2. 访问

浏览器打开：http://localhost:8080

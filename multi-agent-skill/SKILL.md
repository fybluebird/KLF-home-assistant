# 多智能体调度助手

通过呼唤助手名字（如小A、小B、小C）来激活对应的AI助手，每个助手有独立的身份、能力和记忆。

## 功能

- **名字触发**: 喊助手名字即可激活
- **独立记忆**: 每个助手记住自己的对话历史
- **知识共享**: 可以把知识共享给所有助手
- **助手管理**: 添加、删除、列出助手

## 助手

- **小A** - 视觉设计师（主图/视频制作）
- **小B** - 写作助手
- **小C** - 编程助手  
- **小D** - 产品经理

## 命令

| 命令 | 说明 |
|------|------|
| 列出助手 | 显示所有已注册的助手 |
| 添加助手 名字 角色 描述 | 添加新助手 |
| 删除助手 名字 | 删除助手（保留记忆） |
| 记忆 名字 | 查看某助手的历史记录数 |
| 共享知识 内容 | 添加到共享知识库 |

## 使用方式

1. 激活skill后直接喊助手名字，如"小A帮我做个主图"
2. 系统会自动创建/唤醒对应助手并转发消息
3. 助手会记住本次对话，供下次继续
